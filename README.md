# Engenharia de Prompt — Exemplos Simples

Este repositório contém exemplos básicos de **engenharia de prompt** para modelos de linguagem, como o Sabiá-4. O objetivo é demonstrar como construir, testar e otimizar prompts para obter respostas mais precisas e úteis de modelos de IA.

## O que é Engenharia de Prompt?

Engenharia de prompt é o processo de criar, ajustar e refinar instruções (prompts) para que modelos de linguagem entendam melhor o que se espera deles e forneçam respostas mais adequadas.

## Exemplos

Veja alguns exemplos de prompts e suas saídas:

### 1. Prompt Simples

```plaintext
Pergunta: O que é inteligência artificial?
Resposta esperada: Inteligência artificial é a área da ciência da computação que busca criar sistemas capazes de realizar tarefas que normalmente exigiriam inteligência humana.
